# 적용 연습
## 기본

### 한 줄 엔터했을 경우
한 번 엔터하면 / 
문단으로 나눠지지 않음

&nbsp;

### 두 줄 엔터했을 경우 텍스트
두 번 이상 엔터해야 / 

문단으로 나눠짐

&nbsp;

### 줄바꿈
일반 줄바꿈만으로는 공백 입력 X

해결 -> "& nbsp;" 입력

&nbsp;

### 인용구 연습

* 이하 줄바꿈 1회

> 이렇게 하면 인용구
>> 안쪽 인용구
>>> 더 안쪽 인용구

&nbsp;

* 이하 줄바꿈 2회

> 이렇게 하면 인용구

>> 안쪽 인용구

>>> 더 안쪽 인용구

&nbsp;

### 목록 연습

순서 있는 목록

1. 한 줄
2. 엔터
3. 테스트

&nbsp;

1. 두 줄

2. 엔터

3. 테스트

&nbsp;

순서 없는 목록

+ 한 줄
+ 엔터
+ 테스트

&nbsp;

- 두 줄

- 엔터

- 테스트

    (이슈 : 한 줄, 두 줄 모두 줄바꿈이 되는 건 같지만 사이 간격에 차이 발생)

&nbsp;

혼합한 목록

* 이건 별표
    - 이건 마이너스
        + 이건 플러스
            * 이건 다시 별표
                - 이건 마지막 마이너스

## 코드

&nbsp;

### 첫 번째 방법 : 들여쓰기 적용

&nbsp;

이건 일반 텍스트 / 
    이건 엔터 한 번으로 아마 인식 안 될 들여쓰기 / 
이건 일반 텍스트

&nbsp;

이건 일반 텍스트

    엔터 두 번 해줘야 인식됨

이건 일반 텍스트

&nbsp;

### 두 번째 방법 : pre, code

이건 일반 텍스트
<pre>
<code>
어쩌구저쩌구
</code>
</pre>
이건 일반 텍스트

&nbsp;

### 세 번째 방법 : 코드블럭 코드 이용 (일부 적용) 

* 엔터 한 번

이건 일반 텍스트 / 
```화이팅!``` /
이건 일반 텍스트

&nbsp;

* 엔터 두 번

이건 일반 텍스트 /

```화이팅!``` / 

이건 일반 텍스트

&nbsp;

* " ``` " 앞뒤로 엔터 두 번

```

화이팅!

```

(이슈 : 엔터를 한 번만 하면 문장 속 삽입형태가 된다.)
