# 적용 연습
## 기본

### 한 줄 엔터했을 경우
한 번 엔터하면 / 
문단으로 나눠지지 않음

&nbsp;

### 두 줄 엔터했을 경우 텍스트
두 번 이상 엔터해야 / 

문단으로 나눠짐

&nbsp;

### 줄바꿈
일반 줄바꿈만으로는 공백 입력 X

해결 -> "& nbsp;" 입력

&nbsp;

### 인용구 연습

* 이하 줄바꿈 1회

> 이렇게 하면 인용구
>> 안쪽 인용구
>>> 더 안쪽 인용구

&nbsp;

* 이하 줄바꿈 2회

> 이렇게 하면 인용구

>> 안쪽 인용구

>>> 더 안쪽 인용구
